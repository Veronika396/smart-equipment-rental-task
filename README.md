Практична робота на 4 години
Тема: Smart Equipment Rental System
Потрібно створити ASP.NET Core MVC застосунок для оренди техніки.
Сценарій
Компанія має обладнання для оренди:
ноутбуки
камери
проєктори
VR-шоломи
аудіообладнання
Користувач обирає обладнання, створює заявку на оренду, система перевіряє доступність, рахує вартість і створює запис.
Основні вимоги
У застосунку мають бути:
список обладнання
сторінка деталей обладнання
форма створення заявки на оренду
список заявок
сторінка деталей заявки
зміна статусу заявки
Бізнес-правила
При створенні заявки перевірити:
обладнання існує
обладнання активне
дата початку не в минулому
дата завершення більша за дату початку
кількість більша за 0
кількість не більша за доступну
термін оренди не більше 30 днів
Якщо заявка успішна:
створити заявку
порахувати загальну вартість
зменшити доступну кількість обладнання
встановити статус Pending
(Формула TotalPrice = DailyPrice × кількість днів × Quantity)

Статуси заявки
Підтримати статуси:
Pending
Approved
Rejected
Returned
Cancelled
Правила зміни статусів:
Approved можна поставити тільки з Pending
Rejected можна поставити тільки з Pending
Returned можна поставити тільки з Approved
Cancelled можна поставити тільки з Pending або Approved
Якщо статус стає Returned або Cancelled, кількість обладнання має повернутися назад.
DTO
Потрібно створити окремі DTO для різних сценаріїв:
DTO для створення заявки
DTO для списку обладнання
DTO для деталей обладнання
DTO для списку заявок
DTO для деталей заявки
DTO для зміни статусу
Не можна використовувати одну модель для всього.
ViewModel
Створити ViewModel для сторінки створення заявки.
Вона має містити:
інформацію про обладнання
форму заявки
повідомлення про помилку або успіх
Request Pipeline
Додати власну обробку HTTP-запитів:
Middleware для логування:
HTTP method
path
status code
execution time
Middleware для CorrelationId:
створювати унікальний ID для кожного запиту
додавати його в response header
показувати його в логах
3 health endpoint 
Має повертати:
назву застосунку
середовище
поточний UTC час
кількість активного обладнання
кількість заявок зі статусом Pending
UseWhen для модуля оренди:

Для всіх шляхів, які починаються з: /Rental

додати response header:

X-Rental-Module: true
